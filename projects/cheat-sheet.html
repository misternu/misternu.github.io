SQL Cheat Sheet

SELECT - main command used to return data from table, specify which columns

DISTINCT - only return unique data

WHERE - check for a condition and only return that data

AND OR - For adding another condition to where

IN - When using where, you can use in to give it a tuple of values to check

BETWEEN - used to indicate a range of values for a column

Wildcard - percent and underscore are used to create patterns for...

LIKE - Return data that matches the pattern

ORDER BY - Return the data sorted by the column given

GROUP BY - Return the data clumped by the column given

ALIAS - Give the output table a different name

AS - Change the name of the columns in output

SELECT UNIQUE - Return only unique values for the column

INSERT INTO - Insert a row from one table into another

INSERT INTO SELECT - Insert multiple rows from one table into another

UPDATE - Change a record

DELETE FROM - Remove records from a table.
