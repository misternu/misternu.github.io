Inline VS Inline-Block

October 18, 2015

When controlling the flow of elements on a webpage, the css property 'display' can be used to determine if objects show left to right or top to bottom. Most elements are inline by default, and others (like 'p') are block by default. Inline elements are limited in some ways as to how they display, and sometimes those limitations prevent you from making the design you want.

Changing the display property to 'inline-block' will make it so that the objects are still left to right as with inline, but will display with height and other properties as in a block. Inline is appropriate for text and other normally flowing content, and inline-block is more suited to positioning larger page elements like sidebars and navigational buttons.
